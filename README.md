## 📦 安装

在AstrBot的插件市场搜索 `astrbot_plugin_YOTO`，点击安装即可。

## ⚙️ 功能介绍

本插件提供群管、签到、排行榜、每日一言、随机视频等功能，所有功能均可在配置面板中灵活开关和自定义。

### 核心功能

| 功能模块 | 说明 |
|---------|------|
| **管理员设置** | 设置管理员QQ号、群白名单控制 |
| **群管功能** | 全员禁言、禁言/解禁、踢人、拉黑 |
| **签到系统** | 支持24小时制/日期制，固定或随机积分，连续签到奖励 |
| **排行榜** | 积分榜、签到榜、免禁言卡使用榜 |
| **每日一言** | 支持固定文本或一言API |
| **随机小姐姐视频** | 可配置API来源，支持下载发送或转发 |
| **刷屏检测** | 自动检测刷屏行为并禁言 |
| **显示设置** | 菜单/签到/排行榜/个人信息支持图片/文字模式，可自定义背景、字体、颜色、模糊效果 |
| **自定义提示语** | 支持自定义禁言自己时的随机语录 |

## ⌨️ 命令列表

| 命令 | 说明 |
|------|------|
| `菜单` | 查看功能菜单（支持图片/文字两种显示方式） |
| `签到` | 每日签到获取积分 |
| `我的信息` / `个人信息` | 查看个人积分、签到记录等 |
| `积分排行` / `签到排行` / `使用排行` | 查看各类排行榜 |
| `一言` | 获取每日一言 |
| `小姐姐` | 获取随机小姐姐视频（需开启） |
| `禁我 <秒数>` | 禁言自己 |
| `群管配置` | 查看/修改本群配置（管理员） |
| `群管重置 <群号/all>` | 重置配置（管理员） |

*注：命令前缀可在配置中设置，默认为空即直接匹配。*

## 🔧 配置说明

进入插件配置面板进行详细配置，主要配置项如下：

### 管理员设置 (admin)
- **管理员QQ号**: 多个用逗号分隔，如 `10001,10002`
- **群白名单**: 允许使用插件的QQ群列表，空列表表示所有群都允许

### 功能开关 (features)
- **全员禁言**: 开启全员禁言功能
- **禁言/解禁**: 开启禁言相关功能
- **踢人**: 开启踢人功能
- **拉黑**: 开启拉黑功能

### 签到设置 (sign)
- **签到模式**: 24小时制（满24小时可再签）或 日期制（每天0点重置）
- **积分方式**: 固定值 或 范围随机
- **连续签到奖励**: 设置特定天数的额外奖励

### 刷屏检测 (spam)
- **检测消息数量**: 触发检测的消息条数（3-10条）
- **检测时间间隔**: 判断刷屏的时间窗口（0.1-2.0秒）
- **禁言时长**: 刷屏后的自动禁言时间（0-3600秒，0为关闭）

### 显示设置 (display)
- **显示方式**: 菜单/签到/排行榜/个人信息 均可选择 图片 或 文字
- **图片样式**: 支持自定义背景图、字体、标题颜色、正文颜色
- **模糊效果**: 可设置背景图模糊半径（0-5）
- **菜单标题**: 自定义菜单顶部标题和底部居中文本

### 排行榜设置 (rank)
- **显示人数**: 排行榜最大显示人数（5-30人）
- **自定义标题**: 积分榜、签到榜、免禁言卡使用榜的标题

### 每日一言 (daily_quote)
- **来源**: 固定文本 或 一言API
- **API配置**: 支持自定义API地址和JSON字段路径

### 随机小姐姐视频 (girl_video)
- **API地址**: 配置视频源API
- **发送方式**: 直接下载发送文件 或 合并转发 或 发送链接

### 命令与消息 (command / messages)
- **命令前缀**: 设置群内命令前缀（留空直接匹配）
- **自定义语录**: 禁言自己时的随机提示语
